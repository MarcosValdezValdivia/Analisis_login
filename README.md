Analisis de logs de login
Autor: Valdez Valdivia Marcos Yamil

Descripcion 
Este proyecto consiste en el analisis de un archivo CSV que contiene registros de intentos de inicio de sesion (logs de login).
El script esta desarrollado en Python y utiliza la biblioteca "pandas" para procesar y analizar los datos.

El programa realiza:
-Lectura de un archivo CSV
- COnversion de fechas a formato datetime
-Calculo de estadisticas basicas
-Conteo de registros total
-Conteo de intentos fallidos
-Conteo de intentos exitosos
-Calculo de porcentajes de exito y fallo
-Agrupacion de registros por usuario
-Conteo de intentos por pais

Requisitos
- Python 3.x
- Biblioteca pandas
